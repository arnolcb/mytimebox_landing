# My Timebox - Landing Page

The official landing page for **My Timebox**, a minimalist, high-performance timeboxing application designed to eliminate cognitive friction and help you master your day.

## Tech Stack
- **Framework:** [Astro](https://astro.build/) (Static Site Generation)
- **UI:** [React](https://react.dev/) + [Framer Motion](https://www.framer.com/motion/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)

## Development

```bash
# Install dependencies
npm install

# Start local server
npm run dev

# Build for production
npm run build
```
